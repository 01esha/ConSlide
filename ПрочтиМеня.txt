ConSlide

Слайд-шоу для Conky. Цифровая фоторамка для вашего рабочего стола.

Установка

=================================================================

1. Распакуйте архив ConSlide.zip в каталог установки conky.

Например:
unzip /home/user/.conky/ConSlide.zip -d /home/user/.conky/

2. Откройте файл ConSlide в текстовом редакторе.
http://s45.radikal.ru/i107/1702/d4/7ec72882d78b.png

3. Измените минимальный размер окна отображаемого виджета на вашем рабочем столе.

Например:
minimum_size 800 800

Замечание: на системе с окружением xfce и оконным менеджером compiz пришлось еще поправить параметр 
own_window_type override
для скрытия тени рамки виджета. Для других оконных менеджеров такой проблемы не выявил. 

4. Измените время обновления виджета (смены отображаемого изображения) указав время в секундах.

Например, для смены картинки раз в 50 секунд:
${execpi 50 ./slideshow.sh}

!Внимание!
При обновлении изображения происходит его конвертация. 
В зависимости от размера исходного изображения эта процедура кратковременно может занимать до 300 Мб в оперативной памяти. 
Слишком частое обновление изображения может привести к ухудшению производительности вашей системы. 

5. Сохраните изменения в файле ConSlide.

6. Откройте файл slideshow.sh в текстовом редакторе.
http://s013.radikal.ru/i324/1702/64/3418518af0c3.png

7. Измените путь к каталогу, содержащему изображения. В виджете будут отображаться все изображения из данного каталога и его подкаталогов. 

Например:
directory="/home/user/picture/"

8. Измените максимальный размер окна слайд-шоу.

Например:
max_width=750
max_height=520

Обратите внимание — этот параметр должен быть меньше, чем заданный в параметр  minimum_size file файла ConSlide. 

9. Измените цвет и толщину рамки слайд-шоу. Установка толщины равной 0 скроет рамку слайд-шоу. 

Например:
border_color=Gray
border_thickness=15

10. Сохраните изменения в файле slideshow.sh.

11. Запустите виджет ConSlide в conky.
